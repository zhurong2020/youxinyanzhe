你有没有遇到过这样的场景：需要把一段密码、API 密钥或者私密信息发给朋友，却不知道用什么方式最安全？

微信？可能被存档。邮件？可能被扫描。记事本截图？万一对方手机被借用呢？

今天介绍一个我刚部署的小工具：**PrivateBin**，专门解决这个问题。

## 什么是 PrivateBin？

PrivateBin 是一个开源的「加密粘贴板」服务，有几个特别的地方：

1. **零知识加密**：你输入的内容在浏览器里就已经加密了，服务器根本不知道你分享了什么
2. **阅后即焚**：可以设置「阅读一次后自动销毁」
3. **过期时间**：5分钟、1小时、1天、1周...你来定
4. **密码保护**：还可以再加一层密码

## 怎么用？

超级简单，三步搞定：

### 第一步：打开网站，写入内容

访问 PrivateBin 页面，在文本框里输入你要分享的内容。

### 第二步：设置选项

- **过期时间**：建议敏感信息设短一点
- **阅后即焚**：勾选后，对方看完链接就失效
- **密码**：可选，给链接加把锁

### 第三步：发送链接

点击「创建」，复制生成的链接发给对方。

对方打开链接就能看到内容，不需要注册、不需要安装任何软件。

## 为什么比微信/邮件安全？

| 方式 | 问题 |
|------|------|
| 微信 | 消息可能被存档，群聊记录永久保存 |
| 邮件 | 邮件服务商可以读取内容 |
| 截图 | 图片可能被转发，无法控制传播 |
| **PrivateBin** | 端到端加密，阅后即焚，你完全掌控 |

## 适合分享什么？

- WiFi 密码
- 临时账号密码
- API 密钥
- 银行卡号（临时告知家人）
- 任何你不想留下痕迹的文字

## 小贴士

1. **重要信息用「阅后即焚」**：确保对方看完就消失
2. **设置密码**：链接 + 密码双重保护
3. **过期时间别太长**：信息越敏感，过期越短
