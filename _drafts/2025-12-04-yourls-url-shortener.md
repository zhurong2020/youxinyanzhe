你一定用过 bit.ly 或者新浪短链接，把一个很长的网址变成短短的几个字符。但你有没有想过：

- 这些服务会不会收集我的数据？
- 万一它们倒闭了，我的短链接是不是就废了？
- 能不能用自己的域名？

今天介绍一个解决方案：**YOURLS**（Your Own URL Shortener），自己搭建短链接服务。

## YOURLS 是什么？

YOURLS 是一个开源的短链接程序，你可以部署在自己的服务器上，用自己的域名生成短链接。

比如，我把：
```
https://www.example.com/very-long-article-title-about-something
```

变成了：
```
https://mydomain.com/abc123
```

## 自建短链接的好处

| 公共服务 (bit.ly) | 自建 YOURLS |
|-------------------|-------------|
| 用别人的域名 | 用自己的域名，更专业 |
| 数据在别人服务器 | 数据完全自己掌控 |
| 可能被封禁 | 永远可用 |
| 统计功能受限 | 完整的点击统计 |
| 免费版有限制 | 无任何限制 |

## 实际使用场景

### 1. 分享文件下载链接

云盘分享链接又长又丑？缩短它：

- 原始：`https://drive.google.com/file/d/1ABC...XYZ/view?usp=sharing`
- 短链：`https://mydomain.com/report2025`

### 2. 社交媒体个人简介

很多平台简介只能放一个链接，用短链接显得更整洁。

### 3. 线下物料二维码

名片、海报上放一个短链接，用户扫码更方便，你还能统计有多少人扫了。

### 4. 追踪营销效果

给不同渠道生成不同的短链接，就能知道流量从哪里来。

## 管理界面一览

YOURLS 提供一个简洁的管理后台：

- **一键缩短**：粘贴长链接，立即生成短链
- **自定义后缀**：不想要随机字符？可以自己取名，比如 `/resume`
- **点击统计**：每个链接被点了多少次，什么时候点的
- **批量管理**：搜索、编辑、删除已有链接

## 小技巧

1. **取个好记的名字**：比如 `/wechat` 指向你的微信二维码页面
2. **定期检查统计**：了解哪些链接最受欢迎
3. **结合其他工具**：比如把 PrivateBin 的加密链接再用 YOURLS 缩短，双重隐藏

---

如果你也有自己的服务器，不妨试试搭建一个 YOURLS。有了自己的短链接服务，分享链接时会更有掌控感。
